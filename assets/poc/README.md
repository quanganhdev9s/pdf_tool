# POC Test Assets

Các file là dữ liệu tổng hợp, không chứa dữ liệu cá nhân.

- `text_document.pdf`: searchable text, Vietnamese/English.
- `scanned_vi_en.pdf`: image-only PDF for OCR.
- `mixed_rotation.pdf`: mixed page sizes and rotation metadata.
- `existing_crop_box.pdf`: existing crop box.
- `existing_annotations.pdf`: highlight, note, free text, rectangle.
- `forms_and_links.pdf`: AcroForm fields and URL link.
- `image_heavy.pdf`: compression test.
- `large_document.pdf`: 120 pages.
- `password_protected.pdf`: password `poc123`.

Declare these files in `pubspec.yaml`.
