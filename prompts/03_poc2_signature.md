Implement POC 2 — Electronic Signature.

First create and maintain an ExecPlan according to `.agent/PLANS.md`.

Read all relevant project documentation and the completed POC 0 and POC 1 result reports.

Scope:

- Capture a signature using PencilKit.
- Clear and confirm capture.
- Store a reusable signature representation.
- Place it on a selected PDF page.
- Move and resize it.
- Save an editable output.
- Export a flattened output.
- Clearly label it as an electronic signature.

Do not implement certificate-based digital signing.
Do not implement OCR, compression, or unrelated backlog features.
